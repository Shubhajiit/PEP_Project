# Portfolio Website

This is a modern and responsive portfolio website built using HTML and CSS. It showcases various sections such as About, Projects, Skills, Gallery, and Contact.

## Features

- **Responsive Design**: Adapts to different screen sizes for a seamless user experience.
- **Navbar**: Includes links to Home, About, Projects, and Contact sections.
- **Hero Section**: Highlights the portfolio owner's name and profession.
- **About Section**: Provides a brief introduction about the portfolio owner.
- **Projects Section**: Displays a grid of project cards with descriptions.
- **Skills Section**: Lists technical skills categorized into languages, frontend, backend, and tools.
- **Gallery Section**: Showcases images with captions.
- **Contact Section**: Includes a form for users to get in touch.
- **Footer**: Contains social media links and copyright information.

## Technologies Used

- **HTML5**: For the structure of the website.
- **CSS3**: For styling and layout.

## How to Use

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```bash
   cd Task1
   ```
3. Open `index.html` in your browser to view the website.
